# Z-Task 前端项目

基于 React + Vite + Ant Design 的任务管理系统前端。

## 技术栈

- React 18
- Vite 5
- Ant Design 5
- React Router 6
- Axios

## 启动

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build
```

## 项目结构

```
src/
├── assets/          # 静态资源
├── components/      # 公共组件
├── pages/           # 页面组件
│   ├── task/        # 任务管理模块
│   ├── project/     # 项目管理模块
│   └── user/        # 用户管理模块
├── utils/           # 工具类
├── App.jsx          # 根组件
└── main.jsx         # 入口文件
```
