# Z-Task 功能设计文档

## 1. 项目定位

Z-Task 是一个轻量级任务协作管理工具，对标 Teambition / Trello，面向个人和小团队，提供任务看板、列表管理、协作功能。

## 2. 核心功能模块

### 2.1 用户与权限管理

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 用户注册/登录 | 邮箱+密码、支持JWT认证 | P0 |
| 用户管理 | 个人资料、头像、密码修改 | P1 |
| 角色权限 | 管理员、成员、访客 | P1 |

### 2.2 项目空间 (Workspace)

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 创建项目 | 项目名称、描述、图标、可见性(公开/私有) | P0 |
| 项目列表 | 我参与的、我创建的、归档项目 | P0 |
| 成员管理 | 邀请成员、设置角色、移除成员 | P1 |
| 项目设置 | 基本信息修改、归档、删除 | P1 |

### 2.3 任务看板 (Board)

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 看板视图 | 多列表(列)拖拽式看板，类似Trello | P0 |
| 列表管理 | 创建列表、重命名、排序、归档 | P0 |
| 任务卡片 | 标题、描述、截止日期、标签、优先级 | P0 |
| 拖拽排序 | 列表内排序、跨列表移动 | P0 |

### 2.4 任务详情

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 基础字段 | 标题、描述(支持Markdown)、截止日期 | P0 |
| 任务属性 | 优先级(高/中/低)、状态、标签 | P0 |
| 执行者 | 指派给某人、多人协作 | P0 |
| 检查清单 | 子任务清单，可勾选完成 | P1 |
| 附件 | 文件上传(图片、文档等) | P1 |
| 评论 | 任务评论、@提醒 | P1 |
| 活动记录 | 操作日志(谁改了什么) | P1 |

### 2.5 标签管理

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 创建标签 | 标签名、颜色 | P1 |
| 标签筛选 | 按标签筛选任务 | P1 |
| 项目级标签 | 每个项目独立的标签体系 | P1 |

### 2.6 视图模式

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 看板视图 | 默认拖拽看板 | P0 |
| 列表视图 | 列表形式展示任务 | P1 |
| 日历视图 | 按截止日期展示在日历上 | P1 |
| 我的任务 | 聚合所有项目中分配给我的任务 | P1 |

### 2.7 通知与提醒

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 站内通知 | 被指派、被@、评论回复 | P1 |
| 邮件通知 | 重要操作邮件提醒 | P2 |
| 截止日期提醒 | 任务即将到期提醒 | P1 |

### 2.8 搜索与筛选

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 全局搜索 | 搜索项目、任务 | P1 |
| 高级筛选 | 按负责人、标签、截止日期等筛选 | P1 |
| 最近浏览 | 最近查看的任务/项目 | P2 |

## 3. 非功能性需求

| 类别 | 要求 |
|------|------|
| 性能 | 页面加载<2s，看板拖拽流畅 |
| 安全 | JWT认证、接口权限控制、XSS/CSRF防护 |
| 兼容性 | 支持现代浏览器(Chrome/Firefox/Safari/Edge) |
| 响应式 | 支持桌面端优先，适配平板 |

## 4. 技术栈规划

参考 z-config 项目，采用统一技术栈：

| 层级 | 技术 |
|------|------|
| 后端 | Java 8, Spring Boot 2.7.12 |
| 持久层 | MySQL 8.0, MyBatis Plus 3.3.1, Druid |
| 缓存 | Caffeine / Redis(可选) |
| API文档 | Knife4j 3.0.3 |
| 前端 | Vue 3 + Element Plus + Pinia |
| 构建 | Maven |

## 5. 优先级划分

**P0 - 核心功能(MVP)**
- 用户注册/登录
- 项目CRUD
- 看板基础功能(列表、任务卡片、拖拽)
- 任务基础字段(标题、描述、截止日期、优先级、执行者)

**P1 - 重要功能**
- 用户管理、成员邀请、角色权限
- 标签管理、检查清单、评论、附件
- 多种视图(列表、日历)、我的任务聚合
- 通知提醒、搜索筛选

**P2 - 增强功能**
- 邮件通知
- 活动记录/操作日志
- 最近浏览
- 高级权限控制
