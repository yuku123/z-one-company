# 项目总览

## 1. 项目简介

**z-wf** 是一个基于 Spring Boot 的工作流引擎（Workflow Engine）项目，主要用于审批流程管理。该项目采用多模块 Maven 结构，提供完整的工作流定义、流程实例管理、任务处理和审批功能。

## 2. 技术栈

| 技术 | 版本 | 说明 |
|------|------|------|
| Java | 8 | 开发语言 |
| Spring Boot | 2.7.12 | 应用框架 |
| MySQL | 8.0 | 数据库 |
| Druid | - | 数据库连接池 |
| MyBatis-Plus | 3.3.1 | ORM框架 |
| Knife4j | 3.0.3 | API文档（Swagger UI） |
| Lombok | - | 代码生成 |

## 4. 数据库

### 初始化

```bash
mysql -u root -p < z-wf.sql
```

### 主要表结构

- **工作流定义表** - 存储流程定义信息
- **流程实例表** - 存储运行中的流程实例
- **任务/待办表** - 存储待审批任务
- **审计日志表** - 记录审批操作历史

## 5. 快速开始

### 编译构建

```bash
# 完整构建（含测试）
mvn clean package

# 跳过测试快速构建
mvn clean package -DskipTests

# 安装到本地仓库
mvn clean install
```

### Docker 部署

```bash
sh build.sh
```

部署说明：
- 打包应用程序
- 构建 Docker 镜像 `z-wf:latest`
- 部署容器到端口 8081（映射到容器内 8080）
- 通过 `/actuator/health` 端点进行健康检查

## 6. 配置

主配置文件：`z-wf-starter/src/main/resources/application.yml`

```yaml
server:
  port: 8080

spring:
  datasource:
    url: jdbc:mysql://localhost:3306/z_wf?useUnicode=true&characterEncoding=utf8
    username: root
    password: password
    driver-class-name: com.mysql.cj.jdbc.Driver
```

## 7. API 接口

项目提供完整的 **审批中心（Approval Center）** REST API，前端可通过 `/approval-center` 路径调用。

### 接口列表

| 编号 | 接口 | 方法 | 描述 |
|------|------|------|------|
| 001 | `/approval-center/dashboard` | GET | 获取仪表盘统计（待办/已办/发起数量） |
| 002 | `/approval-center/tasks/todo` | GET | 获取待办任务列表（分页） |
| 003 | `/approval-center/tasks/done` | GET | 获取已办任务列表（分页） |
| 004 | `/approval-center/tasks/{taskId}` | GET | 获取任务详情（含表单数据、审批历史） |
| 005 | `/approval-center/tasks/{taskId}/complete` | POST | 完成任务审批（通过/驳回） |
| 006 | `/approval-center/my-processes` | GET | 获取我发起的流程列表 |
| 007 | `/approval-center/processes/{processInstanceId}` | GET | 获取流程实例详情（含审批轨迹） |
| 008 | `/approval-center/processes/start` | POST | 发起流程实例 |
| 009 | `/approval-center/processes/definitions` | GET | 获取可用的流程定义列表 |
| 010 | `/approval-center/processes/{processInstanceId}` | DELETE | 终止/删除流程实例 |

### API 文档

启动应用后，可通过以下地址访问 Swagger 文档：

- **Knife4j（推荐）**: `http://localhost:8080/doc.html`
- **Swagger UI**: `http://localhost:8080/swagger-ui.html`

## 8. 核心概念

### 工作流定义（Process Definition）
流程的模板，定义审批节点、流转规则等。

### 流程实例（Process Instance）
基于工作流定义创建的运行时流程，表示一次具体的审批业务。

### 任务（Task）
流程实例中的待办事项，关联到具体的审批人。

### 审批历史（Approval History）
记录每次审批操作的时间和结果。

## 9. 注意事项

1. **模块依赖关系**: `z-wf-starter` 依赖 `z-wf-core` 和 `z-wf-client`
2. **外部依赖**: 项目依赖内部的 `zifang` 工具库（`util-core`, `util-http`, `util-jdbc`），需确保这些库在本地 Maven 仓库或远程仓库中可用
3. **健康检查**: Docker 部署脚本通过 Spring Boot Actuator 的 `/actuator/health` 端点进行健康检查
