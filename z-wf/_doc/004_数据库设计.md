# 004_数据库设计

## 4.1 表结构总览

| 表名 | 说明 | 关键字段 |
|------|------|---------|
| `ACT_RE_PROCDEF` | 流程定义表 | PROC_DEF_ID_, KEY_, NAME_, VERSION_ |
| `ACT_RE_DEPLOYMENT` | 部署信息表 | ID_, NAME_, DEPLOY_TIME_ |
| `ACT_HI_PROCINST` | 流程实例表 | PROC_INST_ID_, BUSINESS_KEY_, STATE_ |
| `ACT_HI_TASKINST` | 任务实例表 | TASK_DEF_KEY_, ASSIGNEE_, PRIORITY_ |
| `ACT_HI_COMMENT` | 审批历史表 | TASK_ID_, USER_ID_, MESSAGE_ |
| `ACT_HI_ACTINST` | 活动实例表 | ACT_ID_, ACT_TYPE_, ASSIGNEE_ |

## 4.2 流程定义表（ACT_RE_PROCDEF）

| 字段 | 类型 | 说明 |
|------|------|------|
| ID_ | VARCHAR(64) | 主键，流程定义ID |
| REV_ | INT | 版本号 |
| CATEGORY_ | VARCHAR(255) | 分类 |
| NAME_ | VARCHAR(255) | 流程名称 |
| KEY_ | VARCHAR(255) | 流程标识（唯一） |
| VERSION_ | INT | 版本号 |
| DEPLOYMENT_ID_ | VARCHAR(64) | 关联部署ID |
| RESOURCE_NAME_ | VARCHAR(4000) | 流程资源文件名 |
| DGRM_RESOURCE_NAME_ | VARCHAR(4000) | 流程图资源文件名 |
| SUSPENSION_STATE_ | INT | 挂起状态（1=激活，2=挂起） |
| TENANT_ID_ | VARCHAR(64) | 租户ID |

## 4.3 流程实例表（ACT_HI_PROCINST）

| 字段 | 类型 | 说明 |
|------|------|------|
| ID_ | VARCHAR(64) | 主键 |
| PROC_INST_ID_ | VARCHAR(64) | 流程实例ID（唯一） |
| BUSINESS_KEY_ | VARCHAR(255) | 业务键 |
| PROC_DEF_KEY_ | VARCHAR(255) | 流程定义标识 |
| PROC_DEF_ID_ | VARCHAR(64) | 流程定义ID |
| START_TIME_ | DATETIME | 开始时间 |
| END_TIME_ | DATETIME | 结束时间 |
| DURATION_ | BIGINT | 持续时长（毫秒） |
| START_USER_ID_ | VARCHAR(255) | 启动人ID |
| DELETE_REASON_ | VARCHAR(4000) | 删除原因 |
| ROOT_PROC_INST_ID_ | VARCHAR(64) | 根流程实例ID |
| TENANT_ID_ | VARCHAR(64) | 租户ID |
| STATE_ | VARCHAR(255) | 状态 |

## 4.4 任务实例表（ACT_HI_TASKINST）

| 字段 | 类型 | 说明 |
|------|------|------|
| ID_ | VARCHAR(64) | 主键 |
| TASK_DEF_KEY_ | VARCHAR(255) | 任务定义KEY |
| PROC_INST_ID_ | VARCHAR(64) | 流程实例ID |
| NAME_ | VARCHAR(255) | 任务名称 |
| PARENT_TASK_ID_ | VARCHAR(64) | 父任务ID |
| OWNER_ | VARCHAR(255) | 任务拥有者 |
| ASSIGNEE_ | VARCHAR(255) | 办理人 |
| START_TIME_ | DATETIME | 开始时间 |
| END_TIME_ | DATETIME | 结束时间 |
| DURATION_ | BIGINT | 持续时长（毫秒） |
| PRIORITY_ | INT | 优先级 |
| DUE_DATE_ | DATETIME | 截止日期 |
| TENANT_ID_ | VARCHAR(64) | 租户ID |

## 4.5 审批历史表（ACT_HI_COMMENT）

| 字段 | 类型 | 说明 |
|------|------|------|
| ID_ | VARCHAR(64) | 主键 |
| TYPE_ | VARCHAR(255) | 类型（COMMENT） |
| TIME_ | DATETIME | 审批时间 |
| USER_ID_ | VARCHAR(255) | 审批人ID |
| TASK_ID_ | VARCHAR(64) | 关联任务ID |
| PROC_INST_ID_ | VARCHAR(64) | 流程实例ID |
| ACTION_ | VARCHAR(255) | 动作（APPROVE/REJECT） |
| MESSAGE_ | VARCHAR(4000) | 审批意见内容 |
| TENANT_ID_ | VARCHAR(64) | 租户ID |

## 4.6 活动实例表（ACT_HI_ACTINST）

| 字段 | 类型 | 说明 |
|------|------|------|
| ID_ | VARCHAR(64) | 主键 |
| PROC_INST_ID_ | VARCHAR(64) | 流程实例ID |
| ACT_ID_ | VARCHAR(255) | 活动节点ID |
| ACT_NAME_ | VARCHAR(255) | 活动节点名称 |
| ACT_TYPE_ | VARCHAR(255) | 活动类型 |
| ASSIGNEE_ | VARCHAR(255) | 办理人 |
| START_TIME_ | DATETIME | 开始时间 |
| END_TIME_ | DATETIME | 结束时间 |
| DURATION_ | BIGINT | 持续时长（毫秒） |
| TENANT_ID_ | VARCHAR(64) | 租户ID |

SQL 脚本：`z-one-company/z-wf/DB.sql`
