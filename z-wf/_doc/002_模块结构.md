# 模块结构

## 1. 项目概述

`z-wf` 是一个基于 Spring Boot 的工作流引擎项目，采用多模块 Maven 结构，使用 Camunda BPM 作为底层工作流引擎。

## 2. 模块架构

```
z-wf (父项目)
├── z-wf-core      # 核心模块：工作流引擎逻辑
├── z-wf-starter   # 启动模块：Spring Boot 应用
└── z-wf-client    # 客户端模块：外部集成接口
```

## 3. 模块依赖关系

```
z-wf-starter
    ├──► z-wf-core
    └──► z-wf-client

z-wf-client (空壳模块，待扩展)
```

## 4. 各模块详细结构

### 4.1 z-wf-core（核心模块）

**路径**: `z-wf-core/src/main/java/com/zifang/z/wf/core/`

**职责**: 提供工作流引擎的核心实现，基于 Camunda BPM 7.x

**文件结构**:
```
com.zifang.z.wf.core/
├── Main.java                  # 模块入口（占位符）
└── service/
    └── LeaveProcessService.java  # 请假流程服务
```

**技术栈**:
- Camunda BPM Spring Boot Starter 7.19.0（工作流引擎）
- JAXB API 2.3.1（JDK8 兼容性）

---

### 4.2 z-wf-starter（启动模块）

**路径**: `z-wf-starter/src/main/java/com/zifang/z/wf/starter/`

**职责**: Spring Boot 应用入口，提供 REST API 接口

**文件结构**:

```
com.zifang.z.wf.starter/
├── ZWfApplication.java       # Spring Boot 启动类

├── api/                       # 控制器层
│   ├── ApprovalCenterController.java   # 审批中心 API
│   ├── LeaveProcessController.java      # 请假流程 API
│   ├── BaseHealthController.java        # 健康检查
│   └── TodoTaskDTO.java                 # 待办任务 DTO

├── config/                    # 配置类
│   ├── Knife4jConfig.java              # Knife4j 文档配置
│   ├── MybatisPlusConfig.java          # MyBatis-Plus 配置
│   ├── WebConfig.java                  # Web 配置
│   ├── WebMvcConfig.java               # MVC 配置
│   ├── WebLogAspect.java               # 日志切面
│   ├── GlobalExceptionHandler.java     # 全局异常处理
│   ├── TokenInterceptor.java           # Token 拦截器
│   └── TokenContext.java                # Token 上下文

├── dto/                       # 数据传输对象
│   ├── ApprovalRequestDTO.java        # 审批请求
│   ├── StartProcessRequestDTO.java   # 启动流程请求
│   ├── DashboardStatsVO.java          # 仪表盘统计
│   ├── PageResult.java                # 分页结果
│   ├── ProcessDetailVO.java            # 流程详情
│   ├── ProcessInstanceVO.java          # 流程实例
│   ├── TaskDetailVO.java               # 任务详情
│   └── TaskSummaryVO.java              # 任务摘要

└── common/                    # 公共组件
    └── WebExceptionHandler.java       # Web 异常处理
```

**技术栈**:
- Spring Boot 2.7.12
- Spring Boot Starter Web
- MyBatis-Plus Boot Starter 3.5.3
- Druid Spring Boot Starter 1.2.18
- Knife4j OpenAPI3 Starter 4.1.0（Swagger 文档）
- Log4j2（日志）
- MySQL Connector 8.0.33

---

### 4.3 z-wf-client（客户端模块）

**路径**: `z-wf-client/src/main/java/com/zifang/z/wf/client/`

## 7. API 接口概览

审批中心 API 前缀: `/approval-center`

| 类别 | 接口 | 说明 |
|------|------|------|
| 仪表盘 | `GET /dashboard?userId=xxx` | 获取统计数据 |
| 待办 | `GET /tasks/todo?userId=xxx` | 待办任务列表 |
| 已办 | `GET /tasks/done?userId=xxx` | 已办任务列表 |
| 任务详情 | `GET /tasks/{taskId}` | 任务详情（含审批历史） |
| 完成任务 | `POST /tasks/{taskId}/complete` | 审批（通过/驳回） |
| 我的流程 | `GET /my-processes?userId=xxx` | 我发起的流程 |
| 流程详情 | `GET /processes/{processInstanceId}` | 流程详情（含审批轨迹） |
| 启动流程 | `POST /processes/start` | 发起流程 |
| 流程定义 | `GET /processes/definitions` | 可用流程定义列表 |
| 终止流程 | `DELETE /processes/{processInstanceId}` | 终止/删除流程 |
