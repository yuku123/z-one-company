# 001_项目总览

## 1.1 模块定位

z-meta（元数据中心）是企业的数据资产目录，提供数据字典、数据标准、数据血缘追踪能力。所有模块的基础数据口径从这里定义，保证跨模块数据一致性。

## 1.2 愿景

- **数据字典**：字段中英文名、类型、长度、默认值、说明
- **数据标准**：字段填写规范、枚举值列表、校验规则
- **数据血缘**：上游下游字段级血缘，支持可视化
- **数据质量**：完整性/唯一性/一致性校验规则

## 1.3 模块关系

```
z-task（任务元数据：任务优先级/状态枚举）───► z-meta
z-wf（审批流字段：请假天数/审批结果枚举）───► z-meta
z-oss（文件元数据：ContentType 标准化）───► z-meta
z-ctc（用户元数据：账号状态/权限类型枚举）───► z-meta
```
