## 核心功能特性

### 网络通信层 (z-mq-remoting)
- 自定义二进制协议
- 基于Netty的高性能网络通信
- 支持同步/异步/单向调用
- 心跳机制
- 连接管理和流控

### 存储层 (z-mq-store)
- 顺序写入优化
- 内存映射文件（Mmap）
- 同步/异步刷盘策略
- 多文件管理（文件滚动）
- 引用计数资源管理

### 服务层 (z-mq-nameserver)
- Broker注册与发现
- Topic路由管理
- 服务状态监控
- 集群管理

### Broker层 (z-mq-broker)
- 消息接收与存储
- 消息转发与推送
- 消息查询
- 心跳维护

### 客户端层 (z-mq-client)
- 消息发送（同步/异步/单向）
- 消息消费（推/拉模式）
- 负载均衡
- 消费进度管理

## 编译运行

```bash
# 编译整个项目
mvn clean install -DskipTests

# 运行NameServer
cd z-mq-nameserver
mvn exec:java -Dexec.mainClass="com.zifang.z.mq.nameserver.NameServerStartup"

# 运行Broker
cd z-mq-broker
mvn exec:java -Dexec.mainClass="com.zifang.z.mq.broker.BrokerStartup"
```
