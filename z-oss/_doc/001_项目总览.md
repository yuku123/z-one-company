# 001_项目总览

## 1.1 模块定位

z-oss（对象存储服务）为所有模块提供文件存储能力：附件上传、下载、在线预览、CDN 加速。替代直接存数据库的附件方案，支持图片、音视频、文档等多种文件类型。

## 1.2 愿景

- **Bucket 管理**：多 Bucket 隔离（按租户/按用途）
- **文件管理**：上传/下载/删除/复制、移动、批量操作
- **元数据**：文件名、Content-Type、大小、上传者、上传时间
- **访问控制**：签名 URL（临时授权下载）、私有 Bucket
- **预览**：图片缩略图、文档在线预览（规划中）
- **CDN 加速**：对接七牛/阿里云 OSS CDN（规划中）

## 1.3 模块关系

```
z-task（任务附件）───► z-oss
z-wf（审批附件/合同扫描件）───► z-oss
z-ctc（用户头像）───► z-oss
z-meta（数据文件导入）───► z-oss
```

## 1.6 现有代码

**Controller**：`BucketController`（Bucket CRUD）、`ObjectController`（文件 CRUD）、`UserController`（注册/登录）

**实体**：`ZOssBucket`、`ZOssObject`（含 userId、bucketName、objectKey、size、contentType 等）

**状态**：基础文件存取骨架存在，用户注册登录已有实现。CDN、签名 URL、批量操作、在线预览尚未实现。

## 1.7 开发优先级

| 优先级 | 内容 |
|--------|------|
| P0 | Bucket 管理、文件上传下载（本地存储适配）、文件元数据持久化 |
| P1 | 签名 URL 私有访问、用户头像上传、MinIO 替代本地存储 |
| P2 | CDN 接入、文档在线预览、批量操作 |
