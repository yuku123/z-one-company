# 003_接口清单

## 3.1 JobInfoController - 任务管理

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/jobinfo/list` | 获取任务列表 |
| GET | `/jobinfo/{id}` | 获取单个任务详情 |
| POST | `/jobinfo/add` | 新增任务 |
| POST | `/jobinfo/update` | 更新任务 |
| POST | `/jobinfo/remove/{id}` | 删除任务 |
| POST | `/jobinfo/stop/{id}` | 停止任务 |
| POST | `/jobinfo/start/{id}` | 启动任务 |
| POST | `/jobinfo/trigger/{id}` | 触发任务立即执行 |
| GET | `/jobinfo/nextTriggerTime` | 获取下次执行时间 |

## 3.2 执行器回调接口 (Executor Callback)

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/executor/run` | 执行任务 |
| POST | `/executor/kill` | 终止任务 |
| GET | `/executor/log` | 查询执行日志 |
| GET | `/executor/beat` | 心跳检测 |

**说明**：执行器回调接口由执行器侧实现，供调度中心调用。
