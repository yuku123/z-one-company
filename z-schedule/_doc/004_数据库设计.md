# 004_数据库设计

## 4.1 表结构总览

| 表名 | 说明 | 关键字段 |
|------|------|---------|
| `z_schedule_job_info` | 任务信息表 | id, job_group, job_cron, job_desc, author, executor_handler, executor_param, executor_route_strategy, executor_block_strategy, executor_timeout, executor_fail_retry_count, trigger_status, trigger_last_time, trigger_next_time |
| `z_schedule_job_group` | 执行器分组表 | id, app_name, title, order, address_type, address_list |
| `z_schedule_job_log` | 任务调度日志表 | id, job_group, job_id, executor_address, executor_handler, executor_param, executor_sharding_param, trigger_time, trigger_code, trigger_msg, handle_time, handle_code, handle_msg, alarm_status |
| `z_schedule_job_registry` | 执行器注册表 | id, registry_group, registry_key, registry_value, update_time |

## 4.2 任务信息表（JobInfo）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键ID |
| job_group | INT | 执行器主键ID，关联 job_group.id |
| job_cron | VARCHAR(128) | 任务执行CRON表达式 |
| job_desc | VARCHAR(255) | 任务描述 |
| add_time | DATETIME | 添加时间 |
| update_time | DATETIME | 更新时间 |
| author | VARCHAR(64) | 作者 |
| alarm_email | VARCHAR(255) | 报警邮件 |
| executor_route_strategy | VARCHAR(50) | 执行器路由策略（FIRST/LAST/ROUND/CONSISTENT_HASH等） |
| executor_handler | VARCHAR(255) | 执行器任务handler |
| executor_param | VARCHAR(512) | 执行器任务参数 |
| executor_block_strategy | VARCHAR(50) | 阻塞处理策略（SERIAL_EXECUTION/DISCARD/LATER） |
| executor_timeout | INT | 任务执行超时时间，单位秒 |
| executor_fail_retry_count | INT | 失败重试次数 |
| log_id | BIGINT | 调度日志主键 |
| trigger_status | TINYINT | 调度状态：0-停止，1-运行 |
| trigger_last_time | BIGINT | 上次调度时间（时间戳） |
| trigger_next_time | BIGINT | 下次调度时间（时间戳） |

## 4.3 执行器分组表（JobGroup）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键ID |
| app_name | VARCHAR(64) | 执行器AppName |
| title | VARCHAR(64) | 执行器名称 |
| order | INT | 排序 |
| address_type | TINYINT | 执行器地址类型：0=自动注册、1=手动录入 |
| address_list | VARCHAR(512) | 执行器地址列表，多地址逗号分隔 |
| update_time | DATETIME | 更新时间 |

## 4.4 任务调度日志表（JobLog）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键ID |
| job_group | INT | 执行器主键ID |
| job_id | INT | 任务ID |
| executor_address | VARCHAR(255) | 执行器地址 |
| executor_handler | VARCHAR(255) | 执行器任务handler |
| executor_param | VARCHAR(512) | 执行器任务参数 |
| executor_sharding_param | VARCHAR(20) | 执行器分片参数，格式如 1/2 |
| executor_fail_retry_count | INT | 失败重试次数 |
| trigger_time | DATETIME | 调度时间 |
| trigger_code | INT | 调度结果代码 |
| trigger_msg | TEXT | 调度日志 |
| handle_time | DATETIME | 执行时间 |
| handle_code | INT | 执行状态代码 |
| handle_msg | TEXT | 执行日志 |
| alarm_status | TINYINT | 告警状态：0-默认、1-无需告警、2-告警成功、3-告警失败 |

## 4.5 执行器注册表（JobRegistry）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键ID |
| registry_group | VARCHAR(50) | 注册分组 |
| registry_key | VARCHAR(100) | 注册Key（执行器AppName） |
| registry_value | VARCHAR(255) | 注册Value（执行器地址） |
| update_time | DATETIME | 更新时间 |

## 4.6 索引设计

| 表名 | 索引名称 | 索引字段 | 说明 |
|------|---------|---------|------|
| z_schedule_job_info | idx_job_group | job_group | 按执行器分组查询 |
| z_schedule_job_info | idx_trigger_status | trigger_status | 按调度状态查询 |
| z_schedule_job_info | idx_trigger_next_time | trigger_next_time | 查找待调度任务 |
| z_schedule_job_log | idx_job_group_job_id | job_group, job_id | 按任务查询日志 |
| z_schedule_job_log | idx_trigger_time | trigger_time | 按时间查询日志 |
| z_schedule_job_log | idx_handle_code | handle_code | 按执行状态查询 |
| z_schedule_job_registry | idx_registry_key | registry_key | 按执行器查找 |
| z_schedule_job_registry | idx_update_time | update_time | 清理过期注册 |

SQL脚本：`z-one-company/_sql/z-schedule.sql`
