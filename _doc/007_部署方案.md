# 部署方案

## 1. 目录结构

```
z-one-company/
├── bootstraps/
│   ├── z-one-company-main-starter/          # 主启动模块
│   ├── z-one-company-main-starter-frontend/ # React 前端
│   └── z-one-company-ops-starter/           # 运维启动模块
├── _build/                                  # 构建输出目录
│   ├── Dockerfile
│   ├── build-all.sh
│   └── nginx.conf
├── docker/
│   ├── docker-compose.yml
│   ├── mysql/
│   └── redis/
└── _doc/
```

## 2. 服务端口

| 服务 | 内部端口 | 映射端口 | 说明 |
|------|---------|---------|------|
| main-starter（all-in-one） | 8888 | 8080 | 后端主入口 |
| main-starter-frontend（dev） | — | 3000 | 前端开发服务器 |
| z-ctc（独立运行） | 8888 | 8092 | 认证中心 |
| z-wf（独立运行） | 8888 | 8091 | 工作流服务 |
| z-task（独立运行） | 8888 | 8090 | 任务调度 |
| z-mist（独立运行） | 8888 | 8085 | 密钥管理 |
| z-meta（独立运行） | 8888 | 8093 | 元数据 |
| z-config（独立运行） | 8888 | 8848 | 配置中心 |

## 3. 构建命令

### 后端

```bash
# 全量构建（包含测试）
mvn clean package

# 跳过测试构建
mvn clean package -DskipTests

# 安装到本地仓库
mvn clean install

# 启动主应用
cd bootstraps/z-one-company-main-starter && mvn spring-boot:run
```

### 前端

```bash
cd bootstraps/z-one-company-main-starter-frontend
npm install
npm run dev      # 开发服务器（端口 3000）
npm run build    # 生产构建
```

### Docker

```bash
cd _build
docker-compose up -d
```

## 4. 部署模式

### 模式一：All-in-One（开发/演示）

所有后端服务打包在 `z-one-company-main-starter` 中，前端独立部署：

```
┌──────────────────────────────────────────────┐
│              单台服务器 / Mac                 │
│                                              │
│   Port 8080: z-one-company-main-starter      │
│   (z-ctc + z-config + z-task + z-wf 等)    │
│                                              │
│   Port 3000: z-one-company-main-starter-frontend │
│   (React 前端 dev server)                    │
│                                              │
│   Port 3306: MySQL（外部或 Docker）          │
└──────────────────────────────────────────────┘
```

无外部中间件时，服务自带：
- H2 嵌入式数据库（兜底）
- z-cache 内存缓存（兜底）
- z-mq 内存消息（兜底）

## 5. Nginx 配置（生产）

```nginx
server {
    listen 80;
    server_name www.example.com;

    # 前端静态资源
    location / {
        root /usr/share/nginx/html;
        index index.html;
        try_files $uri $uri/ /index.html;
    }

    # 后端 API 代理
    location /api/ {
        proxy_pass http://backend:8080/;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }

    # API 文档
    location /doc.html {
        proxy_pass http://backend:8080/doc.html;
    }
}
```

## 6. 数据库

- **数据库名**：`biz_service`
- **远程地址**：`101.37.80.51:3306`
- **初始化脚本**：`_sql/` 目录下
  - `z-ctc.sql` — 账号、权限、角色、审计日志
  - `z-wf.sql` — 工作流
  - `z-task.sql` — 任务中心
  - `z-config.sql` — 配置中心
