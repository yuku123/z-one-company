# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Common Development Tasks
- `npm run build` - Build the project.
- `npm run lint` - Lint the codebase.
- `npm run test` - Run tests. You can also run a single test using `npm run test:unit -- <test-file>`.

## Code Architecture and Structure
- The main entry point is in `src/index.js`.
- User authentication logic resides in `src/auth`.
- API endpoints are defined in `src/api`.
- UI components are located in `src/components`.
